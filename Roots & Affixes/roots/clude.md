# clude

## Meaning: 关。和 `close` 相近

1. include **[ɪnˈkluːd]** `v` 包含

   ```
   in 里面
   clude 关

   把东西关在里面
   ```

2. exclude **[ɪkˈskluːd]** `vt` 把...排除在外

   ```
   ex 外面
   clude 关

   把东西关在外面
   ```

3. preclude **[prɪˈkluːd]** `v` 防止；阻止

   ```
   pre 预先
   clude 关

   半夜风可能会很大，提前关好窗户防止意外发生
   ```

4. conclude **[kənˈkluːd]** `v` 达成

   ```
   con 一起
   clude 关

   人大闭幕后大家一起关门，达成了一些共识
   ```
