# 缩写

1. `gonna` = `going to`

2. `can't` = `cannot` = `can not`

3. `let's` = `let us`

4. `o'clock` = `of the clock`

5. `I've` = `I have`

6. `wasn't` = `was not`

7. `weren't` = `were not`

8. `sb.` = `somebody`

9. `sth.` = `something`

10. `st.` = `sometime`

11. `sw.` = `somewhere`
