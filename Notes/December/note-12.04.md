# Note 12.04

1. `said the conductor` 和 `the conductor said` 都可以

   ```
   'You must get off here,' the conductor said

   'You must get off here,' said the conductor
   ```

2. `not` 和 `no`

   - `no` 通常和 `n.` 连用，其余的都用 `not`

   - `no + n.` 表示“没有”，`not + 其它词` 表示“不”

   ```
   I have no money

   I do not have money
   ```
