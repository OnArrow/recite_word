# Note 1.08

1. 当单词之间用`-`号连接成复合词时，不要把单词变成复数

   ```
   It's a 24-hour city

   My four-year-old daughter
   ```

2. `How do you do？` 你好

3. `What do you do?` 你是做什么工作的？

4. 有`the`表示地点，没有`the`表示功能

   ```
   go to the school
   去学校
   // 去“学校”这个地点，但目的不一定是上学，我可能是因为找老师，找朋友才去的学校这个地点

   go to school
   上学
   // 学校的功能就是教书育人，所以叫上学

   in the prison
   在监狱里

   in prison
   坐牢

   go to the bed
   走向床边

   go to bed
   睡觉
   ```

5. `a lot` 当作副词时，表示“经常”

   ```
   Do you watch television a lot?
   ```
