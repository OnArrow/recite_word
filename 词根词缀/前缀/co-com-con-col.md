# co - com - con - col

## 含义：聚集，或者说一起，共同

1. company **[ˈkʌmpəni]** `n` 公司

   ```
   一群人聚集在一起干活
   ```

2. collaborate **[kəˈlæbəreɪt]** `v` 合作，协作

   ```
   一起做某件事才能叫合作
   ```

3. combine **[kəmˈbaɪn]** `v` 结合，组合 `n` 联盟

4. connect **[kəˈnekt]** `v` 连接；把...联系起来

5. collect **[kəˈlekt]** `v` 收集；聚集
