# super - over - sur

## meaning: 超过/在...上面

1. surrealism **[səˈriːəlɪzəm]** `n` 超现实主义

   ```
   超越现实的主义
   ```

2. surpass **[səˈpɑːs]** `v` 超过；胜过

3. supermarket **[ˈsuːpəmɑːkɪt]** `n` 超市；超级市场

   ```
   super 超过
   market 集市

   规模超过一般的集市
   ```

4. overlay **[ˌəʊvəˈleɪ]** `n` 遮罩层

   ```
   有一样东西在另一样东西的上方
   ```
