# Lesson 45

1. `a minute` 作时间状语时，表示“一会儿”

   ```
   Give me a minute
   ```

2. `next door` 隔壁

   ```
   Where's Susan?

   She's next door

   Where's your boss's office?

   It's next door
   ```

3. 表示“等我一会”

   ```
   Give me a second

   Wait a minute for me
   ```
