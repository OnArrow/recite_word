2. prompt **[prɒmpt]** `n` 提示；（给演员的）提词 `v` 提示
    ```

    这个程序在必要时将提醒你输入数据
    ```

    ```
    The image is then presented as a captcha in the usual way
    然后这个图片以常规方式显示为验证码
    ```

    ```

    日志缓冲区满了
    ```

5. binary **[ˈbaɪnəri]** `n` 二进制
    ```

    二进制在计算机时代很酷
    ```
