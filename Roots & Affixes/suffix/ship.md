# ship

1. 以 `ship` 结尾的单词

   - 表示某种关系或状态

     ```
     member 成员
     membership 成员身份

     relation 亲戚
     relationship 亲戚关系

     friend 好友
     friendship 好友关系
     ```

   - 表示某种技能

     ```
     salesman 推销员
     salesmanship 推销术

     horseman 骑马的人
     horsemanship 马术

     airman 飞行员
     airmanship 飞行技术
     ```
