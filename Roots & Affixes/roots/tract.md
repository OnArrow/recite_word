# tract

## Meaning: 拉；抽

1. extract **[ˈekstrækt]** `vt` 拔出；抽出 `n` 提取物

   ```
   ex 向外
   tract 抽

   向外抽出植物的汁
   ```

2. contract **[ˈkɒntrækt]** `v` （使）收缩，缩小

   ```
   con 共同；一起
   tract 拉

   很多东西被拉到一起，缩小
   ```

3. distract **[dɪˈstrækt]** `v` 使分心，转移注意力

   ```
   dis 在这里不是否定前缀，通常否定前缀后面跟的是一个完整的单词。在这里表示分开
   tract 拉

   把注意力从某个东西上拉开，分心
   ```

4. abstract **[ˈæbstrækt]** `adj` 抽象的 `n` 抽象

   ```
   tract 抽

   从具体的事物中抽象出来
   ```
