# Lesson 62

1. 表达生病时通常用`have + 病的名称`

   ```
   have an toothache
   牙痛

   have a headache
   头痛

   have an earache
   耳朵痛

   have a temperature
   发烧

   have a cold
   感冒

   have a stomachache
   肚子痛

   I have a temperature
   我发烧了
   ```

2. 当`have`后跟一些名词，往往可以做动词短语

   ```
   have a conversation
   交谈
   // 如果按字面翻译，应该译为“有一个谈话”，这里的“谈话”实际上是一个名词
   // 当前面是`have`时，应该理解为“交谈”，是一个动词短语

   have lunch
   吃午餐
   // 同样，这里不能翻译为“有一个午餐”，应该理解为引申意“吃午餐”这个动词

   have dinner
   吃晚餐
   ```

3. 当表示服用药物时，中文习惯直接说“吃药”，但不能直接翻译成英文`eat medicine`。当用英文表示“吃药”时，应该用`take`

   ```
   take an aspirin
   服用阿司匹林

   take some medicine
   服用一些药
   ```
