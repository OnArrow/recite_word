# nounce

## Meaning: 说

1. announce **[əˈnaʊns]** `v` 宣布；（尤指通过广播）通知

   ```
   a + 双写辅音字母 强调
   nounce 说

   重复说；一再说
   ```

2. denounce **[dɪˈnaʊns]** `v` 谴责，指责

   ```
   de 不足
   nounce 说

   做得不够好，就要挨指责
   ```

3. pronounce **[prəˈnaʊns]** `v` 发（音）；发表意见

   ```
   pro 动态向前
   nounce 说

   向前说
   ```
