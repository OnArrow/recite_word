# Abbreviation

1. `sb.` = `somebody`

2. `sth.` = `something`

3. `st.` = `sometime`

4. `sw.` = `somewhere`

5. `gonna` = `going to`

6. `can't` = `cannot` = `can not`

7. `let's` = `let us`

8. `o'clock` = `of the clock`

9. `I've` = `I have`

10. `he's got` = `he has got`

    ```
    只有当`has got`一起出现时，`has`才能缩写
    ```

    ```
    He has got a car

    He's got a car
    ```

11. `wasn't` = `was not`

12. `weren't` = `were not`

13. `wanna` = `want to`

14. `I'll` = `I will`

15. `won't` = `will not`

16. `shan't` = `shall not`

17. `you'd better` = `you had better`

18. `I'd` = `I would`

    ```
    I would like some tea

    I'd like some tea
    ```
