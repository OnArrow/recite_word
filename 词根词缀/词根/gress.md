# gress

## 含义：走

1. congress **[ˈkɒŋɡres]** `n` 国会，议会

   ```
   con 聚集
   gress 走

   很多人聚集走在一起开会
   ```

2. regress **[rɪˈɡres]** `v` 倒退；回退

   ```
   re 回到原点
   gress 走

   往回走
   ```

3. progress **[ˈprəʊɡres]** `n` 进步，进展

   ```
   pro 动态地往前
   gress 走

   Vue框架在一步一步往前走（发展）
   ```

4. transgress **[trænzˈɡres]** `vt` 逾越；违反

   ```
   trans 转变
   gress 走

   火车走着走着转变到另一条轨道上，逾越
   抽象含义就是本来遵守着某条法令，突然就转变到法令的对立面了
   ```
