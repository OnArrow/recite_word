# Abbreviation

1. `gonna` = `going to`

2. `can't` = `cannot` = `can not`

3. `let's` = `let us`

4. `o'clock` = `of the clock`

5. `I've` = `I have`

6. `he's got` = `he has got`

   ```
   只有当`has got`一起出现时，`has`才能缩写
   ```

   ```
   He has got a car

   He's got a car
   ```

7. `wasn't` = `was not`

8. `weren't` = `were not`

9. `wanna` = `want to`

10. `sb.` = `somebody`

11. `sth.` = `something`

12. `st.` = `sometime`

13. `sw.` = `somewhere`

14. `I'll` = `I will`

15. `won't` = `will not`

16. `shan't` = `shall not`

17. `you'd better` = `you had better`

18. `I'd` = `I would`
