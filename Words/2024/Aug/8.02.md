# 8.02

















































1. downhill **[ˌdaʊnˈhɪl]** `adv` 下坡
    ```
    The rest of way is downhill.
    剩下的路都是下坡路
    ```

2. fur **[fɜː(r)]** `n` 裘衣；毛皮
    ```
    We shouldn't buy fur coats any more.
    我们不应该再买皮大衣了
    ```

3. eager **[ˈiːɡə(r)]** `adj` 热情的；热切的；渴望的
    ```
    He was eager to get into politics.
    他渴望从政
    ```

4. natural **[ˈnætʃ(ə)rəl]** `adj` 天生的，真实的；天然的
    ```
    He was not a natural leader.
    他并非天生的领袖
    ```

5. kinda **[ˈkaɪndə]** `adv` 有点，有几分
    ```
    Write kinda like you talk.
    写出来有点像你在说话
    ```

6. punish **[ˈpʌnɪʃ]** `v` 惩罚，处罚
    ```
    Did you have to punish him very often when you trained him?
    你训练他的时候会经常惩罚他吗？
    ```

7. body **[ˈbɒdi]** `n` 身体；尸体
    ```
    Where did the bullet enter the body?
    子弹从哪个部位穿入身体？
    ```

8. explosion **[ɪkˈspləʊʒ(ə)n]** `n` 爆炸
    ```
    She was blinded in the explosion.
    她在那场爆炸中双目失明了
    ```

9. Benz **[benz]** `n` （汽车品牌）奔驰

10. gaze **[ɡeɪz]** `v` 凝视，注视
    ```
    He gaze settled on her face.
    他的目光落在她脸上
    ```

11. patience **[ˈpeɪʃ(ə)ns]** `n` 耐心
    ```
    I have run out of patience with her.
    我对她失去了耐心
    ```

12. sped **[sped]** `v` （speed 的过去式）

13. rival **[ˈraɪv(ə)l]** `n` 竞争对手
    ```
    Police had to interpose themselves between the two rival groups.
    警方不得不介入这两个敌对团体之间
    ```

14. handsome **[ˈhænsəm]** `adj` （男子）英俊的；（女子）端庄的；漂亮的
    ```
    He used to be so handsome.
    他曾经是那么英俊
    ```

15. granted **[ˈɡrɑːntɪd]** `v` （grant 的过去式） `adv` 的确；想当然
    ```
    We take having an endless supply of clean water for granted.
    我们想当然地认为洁净水的供应无穷无尽而不珍惜
    ```

16. hesitate **[ˈhezɪteɪt]** `v` 犹豫；迟疑
    ```
    If someone asks you out, never hesitate.
    如果有人叫你出去，不要犹豫
    ```

17. excitement **[ɪkˈsaɪtmənt]** `n` 令人激动的事；兴奋
    ```
    The kids were screaming with excitement.
    孩子们兴奋地喊叫着
    ```

18. guilty **[ˈɡɪlti]** `adj` 内疚的；羞愧的
    ```
    I felt a wee bit guilty about it.
    我对此觉得有点惭愧
    ```

19. participle **[pɑːˈtɪsɪpl]** `n` （语法）分词
    ```
    The past participle of regular verbs ends in '-ed'.
    规则动词的过去分词以ed结尾
    ```

20. scornfully **[ˈskɔːnfəli]** `adv` 轻蔑地；蔑视地
    ```
    She screws up his eye scornfully and at him.
    她轻蔑地眯细眼睛盯着他
    ```
