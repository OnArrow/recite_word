1. work **[wɜːk]** `n` 工作；职业 `v` 工作；干活 `adj` 用于工作的；适合工作时穿戴的

   ```
   They don't want to work

   ```

   ```
   今天我得努力工作
   ```

   ```

   她在整理她的床
   ```
