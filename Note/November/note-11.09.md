# Note 11.09

1. 一个动词的过去式可以表示形容词性

   ```
   simplify
   简化
   // 本身是动词

   simplified
   简化了的
   // `simplify`的过去式，这个动作已经完成了。所以是“简化了”，这就有了形容词的意思

   define
   给...下定义

   defined
   轮廓分明的，外形清晰的
   // 因为“给...下定义”这个动作已经完成了，所以此时这个事物就“轮廓分明了，外形也清晰了”

   wash
   清洗

   washed
   洗过的
   // 因为“洗”这个动作已经是过去式了，说明事物已经“洗过了”

   Example:

   Penny: What are you doing Jack?

   Jack: I'm washing my coat

   Two hours later

   Jack's mother: Why is your coat wet Jack?

   Jack: Because it is washed, I washed it two hours ago
   // 第一个“washed”形容词，第二个`washed`动词
   ```
