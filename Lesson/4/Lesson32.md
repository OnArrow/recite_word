1. `tooth` 一颗牙齿 `teeth` 一排牙齿

2. 不可数名词前面不能加`a`

   ```
   Tom's reading a book
   // `book`是可数名词，所以可以在前面加`a`

   Is he drinking water?
   // `water`是不可数名词，所以不能在前面加`a`

   Jack's drinking milk
   // `milk`是不可数名词，所以不能在前面加`a`
   ```
