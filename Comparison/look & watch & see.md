# look & watch & see

1. `look` 通常表示主动地、有意识地"看"，侧重指看的行为

   ```
   I'm looking at the picture
   ```

2. `watch` 也指有意识地看，但往往指仔细地盯着事物的变化

   ```
   I'm watching television
   ```

3. `see` 看到，侧重看到的结果

   ```
   Will you see Jack this afternoon?
   ```

4. 总结

   - 如果是将目光看向某物，用 `look`

   - 如果是眼睛单纯看到的事物，用 `see`
   - 如果是在一段时间内关注某些事物，像是一场比赛或者电视节目，用 `watch`
