# Lesson 104

1. 如果是通过某一个考试，用`pass`。如果是通过某一个科目，用`pass in`

   ```
   I think I passed in English and Mathematics
   // 英语和数学是科目，而不是单纯的某次考试

   I will pass the exam next year
   // 具体的某个考试
   ```

2. 当`paper`翻译为“试卷”时，是 `cn.`

   ```
   There are a lot of papers on the desk
   ```

3. `cheer up` 振作起来
