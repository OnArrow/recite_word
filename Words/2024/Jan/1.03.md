    ```
    你们听了我的笑话从不笑！
    ```

    ```
    This isn't a fire drill
    这不是救火演习
    ```

    ```
    Jazz is mother's milk to me
    爵士乐对于我来说不可或缺
    ```


5. shock **[ʃɒk]** `v` 使震惊；使生气

    ```
    这个消息可把我吓死了
    ```
