# 现在分词

## 含义

- 指一个动作此时此刻正在进行

## 规则

1. 动词后面直接加`ing`

   ```
   read
   reading

   eat
   eating

   climb
   climbing
   ```

2. 以`e`结尾的动词，去掉`e`加`ing`

   ```
   come
   coming

   make
   making

   dance
   dancing
   ```

3. 以`ie`结尾的动词，需要把 `ie` 替换成 `y`，再加 `ing`

   ```
   lie
   lying

   tie
   tying

   die
   dying
   ```

4. 双写最后一个字母加`ing`

   ```
   run
   running

   sit
   sitting

   swim
   swimming

   put
   putting

   shut
   shutting

   begin
   beginning

   rob
   robbing

   forget
   forgetting
   ```
