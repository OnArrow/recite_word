   ```
   Today's game is red team versus blue team
   今天的比赛是红队对抗蓝队
   ```

2. stairs **[steəz]** `n` 楼梯

   ```
   There are no stairs in my house
   我家里没有楼梯
   ```

   ```
   Get this done
   完成这件事
   ```

4. downstairs **[daʊnˈsteəz]** `n` 楼下 `v` 在楼下的
   ```
   Grandmother is downstairs
   奶奶在楼下
   ```
