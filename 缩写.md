# 缩写

1. `gonna` = `going to`

2. `can't` = `cannot` = `can not`

3. `let's` = `let us`

4. `o'clock` = `of the clock`

5. `I've` = `I have`

6. `he's got` = `he has got`

   ```
   只有当`has got`一起出现时，`has`才能缩写
   ```

   ```
   He has got a car

   He's got a car
   ```

7. `wasn't` = `was not`

8. `weren't` = `were not`

9. `sb.` = `somebody`

10. `sth.` = `something`

11. `st.` = `sometime`

12. `sw.` = `somewhere`
