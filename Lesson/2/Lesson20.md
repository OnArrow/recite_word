# Lesson 20

1. 在表示商店不营业的时候，通常用`close`多于`shut`

   ```
   We are closed
   我们关门了

   We are shut
   我们关门了
   ```

   ![close](../../images/Lesson20/open_and_close.png)

2. `Look at them`

   ```
   Look at them
   看他们
   // 根据“主谓宾”结构，`look at`是谓语，谓语后面跟着宾语，`they`的宾语形式是`them`
   ```
