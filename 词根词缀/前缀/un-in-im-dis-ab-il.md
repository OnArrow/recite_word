# un - in - im - dis - ab - il

## 含义：与原单词完全相反，表示否定

1. impossible **[ɪmˈpɒsəb(ə)l]** `adj` 不可能的

   ```
   possible 可能的
   加上了 im 前缀就变为了不可能
   ```

2. disable **[dɪsˈeɪb(ə)l]** `v` 使丧失能力；使无效

   ```
   able 有能力的
   ```

3. untitled **[ʌnˈtaɪtld]** `adj` 无标题的

   ```
   titled 有标题的
   ```
