# 动词的第三人称单数

## 含义

- 如果一个动词跟的主语是第三人称单数（he,she,it）时，动词也要变成第三人称单数形式

## 变化规则

- 直接在后面加 `s`

```
want
wants

like
likes

love
loves
```

- 以 `s`、`x`、`o`、`sh`、`ch` 结尾时，加 `es`

```
do
does

wash
washes

watch
watches
```

- 以辅音字母加 `y` 结尾的，需要把 `y` 替换成 `i`，再加 `es`

```
cry
cries

fry
fries

flies
```

- 不规则情况

```
have
has


```
