# vade

## Meaning: 走

1. invade **[ɪnˈveɪd]** `v` 侵略；入侵

   ```
   in 向内
   vade 走

   美国军队向伊拉克境内走。入侵
   ```

2. pervade **[pəˈveɪd]** `vt` 遍及；弥漫

   ```
   per 每个
   vade 走

   每只猫都在猫岛上到处跑，猫岛上遍及了猫

   某种气味走遍房间的每一个地方。弥漫
   ```

3. evade **[ɪˈveɪd]** `v` 逃避；逃脱

   ```
   e 向外
   vade 走

   在某个责任的重压下往外走，离开这个责任。逃避责任
   ```
