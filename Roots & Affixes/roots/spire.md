# spire

## 含义：气；气体

1. inspire **[ɪnˈspaɪə(r)]** `v` 鼓舞；吸气

   ```
   in 向内
   spire 气

   向身体内吸气，比如紧张的时候深呼吸，鼓舞一下自己
   ```

2. expire **[ɪkˈspaɪə(r)]** `v` 吐气；死亡

   ```
   ex 向外
   spire 气

   向外吐气；
   人吐完身体内的气，也就是死亡
   ```

3. conspire **[kənˈspaɪə(r)]** `v` 密谋；共谋

   ```
   con 一起；共同
   spire 气

   几个人一起串通一气
   ```

4. aspire **[əˈspaɪə(r)]** `v` 渴望；立志

   ```
   a 可以看成冠词
   spire 气

   只要有一口气在，就要做点什么
   ```
