1. both **[bəʊθ]** `pron` 两者 `det` 双方；两者

   ```

   他们两人同时发言
   ```

2. ever **[ˈevə(r)]** `adv` 曾经；从来；在任何时候；始终，一直

   ```

   雨下得比以往都大
   ```

   ```

   我不喜欢夜晚单独外出
   ```


5. gentleman **[ˈdʒentlmən]** `n` （尤指不认识的，或表示礼貌的称谓）先生；彬彬有礼的人

   ```

   他的行为像个真正的绅士
   ```

   ```
   二乘以三等于六
   ```

    ```

    二加四等于六
    ```

    ```
    二乘以三等于六
    ```
