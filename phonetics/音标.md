# 音标

## 音标类型

```
清辅音 [p] [t] [k] [f] [s] [θ] [ʃ] [tʃ] [tr] [ts]
(声带不震动)

浊辅音 [b] [d] [ɡ] [v] [z] [ð] [ʒ] [dʒ] [dr] [dz]
(声带震动)

元音字母

辅音字母
```

## 学习过程中经常碰到的

- `ch` 通常发 `[tʃ]` `[k]`

- `sh` 通常发 `[ʃ]`

- `th` 通常发 `[θ]`

- `se` 通常发 `[z]`

- `ce` 通常发 `[s]`

- `ly` 通常发 `[li]`

- `ce` 在结尾通常发 `[s]`

- `le` 通常发 `[l]`

- `ell` 通常发 `[l]`

  - well

- `ui` 通常发 `[ɪ]`

- `ph` 通常发 `[f]`

- `ou` 通常发 `[aʊ]，[ʌ]`

- `aw` 通常发 `[ɔː]`

- `ge` 通常发 `[dʒ]`

- `ar` 通常发 `[ɑː]`

- `ee` 通常发 `[iː]`

- `oo` 通常发 `[uː]`

- `oa` 通常发 `[əʊ]`

- `ss` 通常发 `[ʃ]`

- `tan` 通常发 `[tən]`

- `tes` 通常发 `[ts]`

- `che` 通常发 `[k]`

- `ful` 通常发 `[fl]`

- `tion` 通常发 `[ʃn]`

- `y` 在结尾通常发 `[i]`

- `e` 在结尾通常不发音

- `tor` 在结尾通常发 `[tə]`

- `ther` 通常发 `[ðə]`
