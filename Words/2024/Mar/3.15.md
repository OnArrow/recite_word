# 3.15












1. several **[ˈsevrəl]** `pron` 几个；数个
    ```
    Several avenues are open to us
    有几个办法可以供我们选择
    ```

2. beyond **[bɪˈjɒnd]** `prep` 超出（范围）；在...较远的另一边
    ```
    Our success was far beyond what we thought possible
    我们的成功远远超出了我们的估计范围
    ```

3. hang **[hæŋ]** `v` 悬挂，吊
    ```
    Don't hang up on me, we must talk!
    别挂掉电话，我们一定得谈谈！
    ```

4. district **[ˈdɪstrɪkt]** `n` 地区
    ```
    They have had a growing influence on the district
    他们对这个地区的影响越来越大
    ```

5. whether **[ˈweðə(r)]** `conj` 是否（表示选择）；无论
    ```
    I asked whether they could change my ticket
    我问他们是否可以给我们换票
    ```

6. honesty **[ˈɒnəsti]** `n` 诚信，诚实
    ```
    Her honesty is beyond question
    她的诚实是毋庸置疑的
    ```

7. handrail **[ˈhændreɪl]** `n` 扶手；栏杆
    ```
    For your safety, please hold the handrail
    为保安全，紧握扶手
    ```

8. determined **[dɪˈtɜːmɪnd]** `adj` 下定决心的；坚定的
    ```
    He's determined to pass ths time
    这一回他决心要考及格
    ```

9. carousel **[ˌkærəˈsel]** `n` 旋转木马；走马灯
    ```
    You'll find your luggage on carousel number three
    你会发现你的行李在3号旋转台上
    ```

10. critic **[ˈkrɪtɪk]** `n` 批评家；评论员
    ```
    He was also for a time an art critic
    他曾有相当长的一段时间是一名艺术批评家
    ```

11. curtain **[ˈkɜːt(ə)n]** `n` 窗帘，门帘
    ```
    She pulled the curtain aside
    她把窗帘拉向一边
    ```

12. foreigner **[ˈfɒrənə(r)]** `n` 外国人
    ```
    You don't sound like a foreigner talking English
    你听起来不像外国人在说英语
    ```

13. sympathetic **[ˌsɪmpəˈθetɪk]** `adj` 表示同情的；惺惺相惜的
    ```
    You have to be firm, but at the same time you should try and be sympathetic
    你必须要严格，不过也应尽量怀有同情心
    ```

14. modern **[ˈmɒd(ə)n]** `adj` 当代风格的；摩登的，时髦的
    ```
    I don't really go for modern art
    我并不是很喜欢现代艺术
    ```

15. ellipsis **[ɪˈlɪpsɪs]** `n` 省略号
    ```
    To browse to the location of the file, click the ellipsis button
    如要浏览文件位置，请单击省略号
    ```

16. upset **[ˌʌpˈset]** `adj` 心烦意乱的；不安；不满意
    ```
    You looked pretty upset just now
    你刚才看上去心烦意乱的
    ```

17. mad **[mæd]** `adj` 发疯的 `n` 字幕版
    ```
    You must be mad to risk it
    你去冒这种风险，简直是疯了
    ```

18. wicked **[ˈwɪkɪd]** `adj` 邪恶的；很坏的
    ```
    A wicked light flamed in his eyes
    他的眼睛里闪着邪恶的光芒
    ```

19. pretend **[prɪˈtend]** `v` 假装，伪装
    ```
    Don't be surprised if I pretend not to recognize you
    要是我假装不认识你，你别觉得奇怪
    ```

20. appreciate **[əˈpriːʃieɪt]** `v` 欣赏，鉴赏；感激
    ```
    Anyone can appreciate our music
    任何人都能欣赏我们的音乐
    ```
