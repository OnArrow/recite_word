# claim

## Meaning: 喊

1. proclaim **[prəˈkleɪm]** `v` （公开）声明；（正式）宣告

   ```
   pro 动态向前
   claim 喊

   向前大声喊
   ```

2. reclaim **[rɪˈkleɪm]** `v` 要求归还

   ```
   re 回到原点
   claim 喊

   大声喊要某物回到原点
   ```

3. acclaim **[əˈkleɪm]** `v` 赞扬

   ```
   acc 双写辅音字母表示强调
   claim 喊

   喊的色彩更浓烈，喊得很激动
   ```

4. exclaim **[ɪkˈskleɪm]** `v` 呼喊；惊叫

   ```
   ex 向外
   claim 喊

   向外大声地喊
   ```
