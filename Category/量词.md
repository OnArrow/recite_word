# 量词

1. `a loaf of` 一个

2. `a bar of` 一条

3. `a bottle of` 一瓶

4. `a cup of` 一杯

5. `a glass of` 一杯

6. `a pound of` 一磅

7. `half a pound of` 半磅

8. `a quarter of` 四分之一

9. `a tin of` 一听

10. `a can of` 一罐

11. `a pot of` 一壶

12. `a slice of` 一片

13. `a pint of` 一瓶脱（液体单位，约等于半升）

14. `a pair of` 一副；一双

    ```
    I need a new pair of shoes

    ```

15. `a packet of` 一袋；一包

    ```
    He went out to buy a packet of cigarettes
    ```
