# 10.13

   ```
   I had to choose between the two jobs
   我得在两份工作之间做出选择
   ```

   ```

   该审判之后他拒绝发表评论
   ```

   ```

   他出去买了一包香烟
   ```

   ```

   我们明天去野餐
   ```

   这是全年赛马日程表中最重要的几个星期之一
   ```

   ```

   ```

7. ground **[ɡraʊnd]** `n` 地面；土地，土壤；话题，知识领域 `adj` 切碎的，磨碎的

   我发现她躺在地上

    ```

    她的表情突然变了

    这是我得到过的最好分数
