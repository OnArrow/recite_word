# sub - sup - de -under

## Meaning: 不足/在...下面/向下

1. decrease **[dɪˈkriːs]** `v` 减少；降低

   ```
   某样东西应该达到100%的，但是它不足以达到
   ```

2. degrade **[dɪˈɡreɪd]** `v` 贬低；降级

3. subway **[ˈsʌbweɪ]** `n` 地铁

   ```
   在地下的路
   ```

4. subtract **[səbˈtrækt]** `vt` 减去

   ```
   数字往下减少
   ```

5. decrement **[ˈdekrɪmənt]** `n` 自减
